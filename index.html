K6BP Rigcontrol Web Firmware Installer

To install the Rigcontrol firmware, connect the Rigcontrol board to a computer via USB, and run a web browser on that same computer.

You must use a Chrome or Edge web browser.
You must use Windows, MacOS, or Linux.
And you must use the https version of this page, plain http won't work. https://perens.com/static/Rigcontrol/

Connect your Rigcontrol board via USB. Use the top USB connector on the Rigcontrol board. That will provide both data and power.

When the installer asks what serial port you wish to use, select the one with a CP201 USB to UART Bridge Controller, that is the USB chip on the Rigcontrol board.

If the installer asks if you want to erase the device, say yes if you are installing K6BP Rigcontrol for the first time, or if there is a different program currently installed on the device. This will erase any saved settings.

Push the button below to start the installer.

When installation is done, use the menu option to configure the board's WiFi parameters.

Troubleshooting

  • If you can't access the card, unplug and reconnect its USB interface. Sometimes, it's necessary to remove and restore power to the USB hub the card is connected to.
  • The first time you run this, you may have to grant yourself permission to open serial ports on your computer. You'll only need to do this once per user and computer.

    On Linux use the command

    sudo adduser your-user-name dialout
    Then log out, and in again.

    On MacOS, use these commands:

    sudo dseditgroup -o edit -a your-user-name -t user wheel
    sudo dseditgroup -o edit -a your-user-name -t user admin
    Then log out, and in again.

    If you are running Windows, the system may ask for permission to access the serial device. Give it.

    Restricting permission to access serial ports was important when they were used for dial-out telephone modems. It's still there today.

See the Github page for information on the Rigcontrol project.

The "Rigcontrol" board is actually an ESP32-Audio-Kit board made by AI Thinker. It becomes K6BP Rigcontrol when our firmware is installed.
